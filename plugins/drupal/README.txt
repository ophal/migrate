## Migration to Ophal helper for Drupal

This folder (plugins/drupal) contains a Drupal module in the several supported
versions. Drupal 6.x is the only supported version at this time.

### How to install?

1. Open the file ophal_export/ophal_export.module
2. Find the text "YOUR TOKEN HERE" and replace it with a long passcode
3. Instal ophal_export into your drupal site
